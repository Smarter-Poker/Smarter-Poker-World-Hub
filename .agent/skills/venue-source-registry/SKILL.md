This skill provides the **authoritative source of truth** for all 777 poker venue scraping operations.

1. **PokerAtlas** - Preferred for ~400 major venues (standardized format)

## üîß SCRAPER PATTERNS

### PokerAtlas HTML Structure
```html
<table class="tournament-schedule">
  <tr class="tournament-row">
    <td class="day">Monday</td>
    <td class="time">7:00 PM</td>
    <td class="buyin">$125</td>
    <td class="name">NLH Deep Stack</td>
    <td class="gtd">$5,000 GTD</td>
  </tr>
</table>
```

### Venue Website Patterns
Most venue websites use one of these formats:
1. **HTML Table** - Similar to PokerAtlas
2. **Card Grid** - CSS cards with tournament info
3. **Calendar View** - Event calendar with modals
4. **PDF Schedule** - Downloadable PDF (requires OCR)

### Search Query Templates
```
"{Venue Name}" "{City}" poker tournament schedule site:{domain}
"{Venue Name}" daily poker tournaments
"{Venue Name}" poker room schedule 2026
```

## ‚ö†Ô∏è VENUES REQUIRING SPECIAL HANDLING

### PDF Schedules Only
- Some smaller venues only publish PDF schedules
- Require manual extraction or OCR
- Flag in database: `schedule_format = 'pdf'`

### No Online Presence
- ~27 venues have minimal/no online schedule
- Call venue directly for schedule
- Flag in database: `schedule_format = 'call_venue'`

### Seasonal/Temporary Closures
- Track `is_active` status
- Check quarterly for reopenings
