# No Emoji Button Standard

## Rule
**NEVER use emojis inside button elements.** All buttons must use clean, text-only labels.

## Applies To
- All `<button>` elements
- All `<Link>` components styled as buttons
- Tab buttons
- Action buttons
- Form submit buttons

## Rationale
Clean, professional Facebook-style UI without emoji clutter.
