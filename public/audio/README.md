# Cinematic Intro Audio

This folder contains the audio file for the Smarter.Poker cinematic intro.

## How to Add Your Premium Audio

1. **Get a professional cinematic intro sound** (4 seconds or less)
   - Recommended sources:
     - [Mixkit](https://mixkit.co/free-sound-effects/logo/) - Free royalty-free
     - [Uppbeat](https://uppbeat.io/sfx/tag/logo) - Free with attribution
     - [Epidemic Sound](https://www.epidemicsound.com/) - Premium subscription
     - Custom composition from a producer

2. **Name the file**: `cinematic-intro.mp3`

3. **Place it in this folder**: `/public/audio/cinematic-intro.mp3`

4. **The intro will automatically use your audio file**

## Audio Specifications

- **Duration**: 3-4 seconds (matches visual intro timing)
- **Format**: MP3 (recommended) or WAV
- **Style**: Cinematic impact, logo reveal, epic brass/synth
- **Loudness**: -14 LUFS (streaming standard)

## Fallback

If no local file is found, the intro will automatically use a royalty-free 
cinematic impact from Mixkit CDN as a fallback.
